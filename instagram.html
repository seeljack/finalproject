Instagram Clone

This project is an Instagram Clone, focused on client-side dynamic pages. It leverages JavaScript, React, and REST APIs to deliver a smooth and interactive user experience. The project demonstrates features such as user authentication, posting, commenting, liking, and infinite scrolling.

How It Works

  • Client-Side Rendering: The application uses JavaScript to dynamically update the DOM without reloading the page.
  • REST API: Backend functionality is accessed via a REST API for posts, comments, and likes.
  • AJAX Calls: JavaScript fetches data asynchronously, enabling features like liking posts or adding comments in real-time.
  • Infinite Scrolling: Posts are loaded dynamically as the user scrolls, ensuring a seamless browsing experience.
  • User Authentication: Login and session management are handled via secure tokens.

Features

  • Posting images and captions
  • Commenting on posts
  • Liking posts with real-time updates
  • Infinite scrolling for a continuous feed
  • Responsive UI for desktop and mobile

Technologies Used

  • Frontend: React, JavaScript, HTML, CSS
  • Backend: Python, Flask, SQLite
  • Testing:Pytest
  • Deployment: AWS
