Wall Street Bets Stock Simulator

What this does:

User inputs a date. The program then takes the percent change of the top wall street bet stocks under the conditions the user put in the function 'wall_street_bet_stock_from_previous_day'. It takes the stock price of each stock for a period of {test_length}, starting at the date after the date inputed. It will then compare these stocks against the vanguard 500, to see how it compares. For Example: User entered: 2022-03-04. Test_Length = 10. Program gets top stocks from wall street bets from dates (2022-03-05-2022-03-15). The number of stocks it takes into account, number of comments the stocks chosen have, ETC, can be changed in the function 'wall_street_bet_stock_from_previous_day'. It then uses those conditions to see the average change all the stocks had during that time period, then compares it with the controll (vanguard 500). Furthermore, you can also test stocks for the current day, and create databases using SQLITE.

Unfortunatly you must download the code from the github link below and run it locally but I am currently working on a way to deploy this application with flask/Heroku.

Moving Forward:

Some interesting things to take into account with this is while Wall Street Bets may not be the most promising you can use this same idea for other APIs. You can look at news/stock sentiment, cashflow, earnings, etc for top stocks and run the program a large amount of times to get a p score (when comparing it to other indicators) and find which conditions lead to the best results.
